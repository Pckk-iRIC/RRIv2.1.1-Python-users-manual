# 用語集

- **ASC**: ASCII グリッド形式（ヘッダー + 数値配列）
- **out**: RRI 本体の出力ディレクトリ
- **prefix**: 出力ファイル名の先頭記号（例: `hf_`, `hs_`）
- **NODATA**: 欠損値を表す定数
- **fine 格子**: 通常格子より細かい解像度の格子
- **RAW (.out)**: RRI 本体が出力するヘッダー無しの数値グリッド
- **header template**: ヘッダー付与に使う参照 ASC
- **I/J**: RRI のセル座標（1始まり）
- **EPSG**: 座標参照系の識別番号
- **landuse**: 土地利用区分コード（整数）
- **grad**: 勾配（平均勾配、sin θ）
- **DEM**: 標高（Digital Elevation Model）
- **dir**: 流向（8方向）
- **acc**: 累積流量（flow accumulation）
