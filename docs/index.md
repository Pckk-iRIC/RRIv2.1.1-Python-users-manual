# このマニュアルについて

本書は **RRI v2.1.1 Python版** の利用者向けマニュアルです。GUI/CLI を使って **入力データ作成 → 解析補助 → 後処理可視化** を実行できるよう、非エンジニアでも迷わないことを重視しています。

- 対象バージョン: v1.0.0
- 対象読者: RRI の利用者（Python/CLI に不慣れな方を含む）
- 対象範囲: `main.py` から起動できる機能全般 + GUI 操作

> 注意
> - 本書は **ユーザー向け** のみを扱います。開発者向けの内部仕様は含みません。
> - 画像は `docs/images/` 配下で管理します（今後追加予定）。
